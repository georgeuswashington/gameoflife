# Survive Life: Neon Reboot

Минималистичная браузерная life-sim игра на русском языке.

## Что уже есть (MVP-каркас)
- Профили игроков (несколько сохранений).
- 3 сложности старта: Легко / Базово / Сложно.
- Игровое время в реальном времени (1 сек = 1 игровая минута) с ускорением x1/x2/x5/x10.
- Базовые шкалы 0–1000: голод, энергия, настроение, здоровье, стресс, гигиена, комфорт.
- Локации: дом, работа, магазины, банк, рынок труда, медицина, админ-панель.
- Действия дома: сон, душ, чистка зубов, еда, посуда, тренировка, прогулка.
- Работа: 3 стартовые профессии, оплата за действия, репутация, попытка повышения.
- Экономика: ежемесячные списания, зарплата, кредит 18% годовых, вклад 8% годовых.
- Случайные события и лента уведомлений.
- Сохранение в localStorage + экспорт/импорт JSON.

## Запуск
Откройте `index.html` в браузере.
